충청톤 팀3 — KNUT

AI 기반 협력 일정 관리 플랫폼 · 백엔드 개발 담당

Spring Boot 3.5.3 Java 21 MySQL JWT Google Gemini API Docker

프로젝트 개요

KNUT는 충청톤 해커톤에서 개발한 AI 기반 협력 일정 관리 플랫폼입니다. 사용자가 일정을 등록하면 Google Gemini AI가 일정 데이터를 분석해 실행 가능한 인사이트와 리포트를 제공합니다. 친구 관계 기능을 통해 팀원과의 협력도 지원합니다.

내가 담당한 것

🔐

JWT 인증 시스템

Spring Security와 JJWT 라이브러리를 활용해 로그인/회원가입 API와 JWT 토큰 발급·검증 로직을 구현했습니다. 토큰 유효기간은 24시간으로 설정했습니다.

📅

일정 CRUD API

일정 생성, 조회(날짜/범위별), 우선순위 수정, 완료 처리, 삭제 등 일정 관련 전체 REST API를 설계하고 구현했습니다. JPA를 사용해 데이터베이스와 연동했습니다.

🤖

AI 일정 분석 (Gemini API)

Google Gemini 2.0 Flash 모델을 연동해 사용자의 일정 데이터를 분석하고 3가지 실행 가능한 인사이트를 제공하는 기능을 구현했습니다. API 실패 시 통계 기반 폴백 응답도 처리했습니다.

👥

사용자·친구 관계 API

회원가입, 이메일 중복 확인, 프로필 조회·수정, 계정 삭제 API와 사용자 간 친구 관계 관리 기능을 구현했습니다. Google OAuth 소셜 로그인도 지원했습니다.

🐳

Docker 배포 환경 구성

OpenJDK 21 기반 Dockerfile을 작성하고, Docker 환경용 application-docker.yml을 별도 구성해 개발과 배포 환경을 분리했습니다.

🏗️

레이어드 아키텍처 설계

Controller - Service(인터페이스/구현 분리) - Repository 레이어드 아키텍처로 프로젝트 구조를 설계해 유지보수성과 확장성을 고려했습니다.
